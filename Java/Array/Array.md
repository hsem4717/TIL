# 04-1. Array

<h3>배열</h3>

-같은 타입의 데이터를 연속된 공간에 나열하고 각 데이터에 인텍스를 부여해놓은 자료구조   
-같은 타입의 데이터만 저장 가능   
-한번 생성된 배열은 길이를 늘리거나 줄일 수 없음   
-배열 변수는 참조 변수에 속함
