`public class Hello`부분을 클래스 선언부라고 함   
`public class`: 공개 클래스   
`Hello`: 클래스 이름

`public static void main(String[] args)`부분을 메소드 선언부라고 함   
`main`:메소드 이름

바이트 코드 실행   
->`main()`메소드를 찾아 블록내부 실행   
그래서 `main()`메소드를 진입점이라고 부름
